How to get a Bitpanda API key

  1. Open https://web.bitpanda.com/apikey.
  2. Give the key a name, like PandaGainz, and mark all scopes (Trading, Transaction, Balance).
  3. Click "Generate new API key" and copy the key value.
  4. Go back and enter your API key to generate your report.

What is PandaGainz?

  • It's fully open source.
  • It's free to use.
    • In the future users will be able to donate to unlock extra features.
  • Calculations are done using the FIFO (first in first out) method.
  • Happy? Generate your report now!

How secure is it?

  • It does NOT store or share API keys provided by users.
    • If you mark the "Remember API key" check box, it will save the encrypted API key on a browser cookie.
  • It does NOT store any generated reports without the user's consent.
    • In the future users will be able to opt-in to have their reports stored on a database for future reference.

More technical details please

  • Written mostly in TypeScript, running on Node.js.
  • Tastes like vanilla. No React, no Vue, no Angular, no jQuery.
  • Service hosted on Google Cloud, but dockerized it can run almost anywhere.
